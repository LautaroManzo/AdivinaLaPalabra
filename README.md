## 🚀 Tecnologías utilizadas

### 🏗️ Backend
- **.NET 8 / .NET Core** - C#
- **Arquitectura MVC**
- **Entity Framework**
- **LINQ**
- **PostgreSQL** (Base de datos)
- **Hangfire** (Manejo de tareas en segundo plano)
- **SignalR** (Comunicación en tiempo real)

### 🎨 Frontend
- **JavaScript / jQuery**
- **Bootstrap**
- **CSS**
